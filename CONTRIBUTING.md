## Submitting completed sheets

The main way to contribute is by submitting your own completed Falsifiability Sheets:

1. Download a template from [`templates/`](templates/)
2. Complete the full workflow including R1 and R2 reviews
3. Read the [Zenodo record abstract](https://zenodo.org/records/18518231) for submission instructions
4. Submit your finished sheet (PDF) to the Zenodo community

## Reporting issues

If you find a problem with the templates, quick-start guide, or documentation:

- Open a [GitHub issue](https://github.com/mlge9900-crypto/echoes-of-the-script-openlab/issues) describing what's wrong or unclear
- Include which file and section you're referring to

## Suggesting improvements

Ideas for improving the Falsifiability Sheet workflow, adding new templates, or expanding the worked examples are welcome. Open an issue to start the discussion before submitting a pull request.
