# Security Policy

## Reporting a Problem

Please report serious issues to: mlge9900@gmail.com

Include:
- what file or page is affected
- what went wrong
- screenshots if possible
- steps to reproduce the issue

## What to Report

- broken or misleading files
- evidence integrity problems
- citation or provenance errors
- AI output issues
- anything that could make the toolkit unreliable

## Response Goal

We aim to review reports within 7 days and issue a versioned fix when needed.
